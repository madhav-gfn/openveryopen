# OpenVeryOpen

A collection of sample code and tutorials for learning OpenCV in C++.

## Overview

This repository contains practical examples and tutorials demonstrating various OpenCV functionalities, from basic image processing to advanced computer vision techniques. Each sample is designed to help you understand and implement common CV operations.

## Getting Started

### Prerequisites

- C++ compiler (GCC, Clang, or MSVC)
- OpenCV 4.0+ installed
- CMake 3.10+ (optional, for building)

### Installation

1. **Install OpenCV** (if not already installed):
   - [OpenCV Installation Guide](https://docs.opencv.org/master/d0/d3d/tutorial_general_install.html)

2. **Clone this repository**:
   ```bash
   git clone https://github.com/yourusername/openveryopen.git
   cd openveryopen
   ```

3. **Build the samples**:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

## Contents

The repository includes tutorials on:

- **Image Basics** - Reading, writing, and displaying images
- **Image Processing** - Filtering, edge detection, transformations
- **Feature Detection** - Corners, edges, and keypoint detection
- **Object Detection** - Using pre-trained models and classifiers
- **Video Processing** - Working with video files and cameras
- **Drawing & Annotations** - Adding shapes and text to images

## Running the Samples

After building, run any sample:

```bash
./build/sample_name
```
